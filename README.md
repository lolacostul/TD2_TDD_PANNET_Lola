Install from : https://www.python.org/downloads/

### Unittest
This repo uses Unittest.

### Cloning repo
With HTTPS:
```sh
git clone https://github.com/TPtestLogiciel/TP_TEST_LOGICIEL.git
``` 
or with SSH:
```sh
git clone git@github.com:TPtestLogiciel/TP_TEST_LOGICIEL.git
```
